# Pequenos Construtores

Jogo cristão educativo de construção e exploração em Godot.

## Estado atual

- Menu inicial.
- Escolha de personagem.
- Player 3D controlável com WASD, pulo e câmera com mouse.
- Placeholder humano com variações visuais e animação simples de caminhada.
- Primeiro mapa jogável: vale bíblico/desértico com área central plana para futuras construções.

## Stack

- Godot Engine 4.x
- GDScript
- Plataforma inicial: PC

## Como executar

1. Abra `project.godot` no Godot.
2. Rode o projeto.
3. Clique em `Novo Jogo`.
4. Escolha um personagem.
5. Use `WASD`, `Espaço` e `ESC`.
